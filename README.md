# Ini Lim × d3pot-wrks pitch

A lightweight static portfolio pitch.

## Run locally
Open `index.html` in a browser.

## Deploy
Upload this folder to Vercel, Netlify, or any static host.

## Files
- `index.html` — content and layout
- `style.css` — styling and colour palette
- `script.js` — interactions and role content

- `#BBE6E4`
